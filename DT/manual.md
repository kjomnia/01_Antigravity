# Digital Twin (Smart Station Manager) 사용 설명서

## 1. 프로그램 개요
본 프로그램은 국사 내 랙(Rack) 및 설비 정보를 관리하고, QR/바코드를 생성하여 라벨 프린터로 출력하거나 엑셀로 저장하는 기능을 제공합니다.

---

## 2. 설치 및 실행 (필수 확인)

### 2.1. 설치 방법
본 프로그램은 별도의 설치 과정 없이 사용할 수 있는 **무설치(Portable)** 버전입니다.
1. 배포된 **`Digital Twin.zip`** 파일을 다운로드합니다.
2. 원하는 위치(예: 바탕화면 등)에 압축을 풉니다.

### 2.2. 실행 방법
1. 압축 해제된 폴더 내의 **`DigitalTwin.exe`** 파일을 더블 클릭하여 실행합니다.
   > [!IMPORTANT]
   > `Application` 폴더에는 프로그램 구동에 필요한 파일들이 들어있습니다. 절대로 삭제하거나 이동하지 마세요.

### 2.3. 문제 해결 (실행이 안 될 때)
#### 보안 경고창이 뜨는 경우 (Windows Defender)
1. **Windows의 PC 보호** 파란색 창이 나타나면 **[추가 정보]**를 클릭합니다.
2. 하단에 생기는 **[실행]** 버튼을 클릭하면 정상적으로 실행됩니다.
   *(최초 1회만 수행하면 됩니다.)*

---

## 3. 주요 기능별 사용법

### 탭 1: QR바코드 (현장 실사 입력)
현장에서 랙 정보를 입력하고 QR 코드를 생성하는 메인 화면입니다.

**1. 국사 선택**
- 우측 상단의 검색창에서 국사명을 검색하고 클릭하여 선택합니다.
- 선택된 국사의 정보(팀명, 국사코드 등)가 우측 패널에 표시됩니다.

**2. 랙 모델 조회**
- 상단 패널에서 **상징, RU/Height, Width, Depth**를 선택하여 랙 모델을 조회합니다.
- 일치하는 모델이 있으면 모델 ID가 초록색으로 표시됩니다.
- 입력된 조건에 따라 가능한 규격 목록이 자동 필터링됩니다.

**3. 데이터 입력 (그리드)**
- **구역/위치**: 구역(Zone), 랙 위치(행)을 입력하면 다음 위치(열) 자동으로 값이 채워집니다.
- **자동 완성**: 랙 ID, 랙 모델, 랙 위치에 대한 QR 코드가 입력된 정보를 바탕으로 자동 생성됩니다.
- **랙 모델 일치화**: 구역(Zone), 랙 위치(행), RU&H(mm),	W (mm),	D (mm) 입력시 랙 모델 ID 와 일치하면 녹색, 불일치시 빨강색으로 표시됩니다. 
- **행 추가**: Enter 또는 방향키(↓)를 누르면 자동으로 다음 행이 추가됩니다.

**4. 출력 및 저장**
- **출력**: [출력] 버튼을 클릭하면 라벨 프린터로 QR 코드를 인쇄합니다.
- **저장**: [저장] 버튼을 클릭하면 입력된 내용을 엑셀 파일로 다운로드합니다.
- **초기화**: [초기화] 버튼으로 모든 입력 내용을 지울 수 있습니다.

### 탭 2 & 3: 선로등록 / 장비WDM
- **선로등록**: 선로 관련 정보를 입력하고 관리합니다.
- **장비WDM**: WDM 장비 관련 정보를 입력합니다.
- 사용 방법은 'QR바코드' 탭의 그리드 입력 방식과 유사합니다.

### 탭 4: 관리자 모드 (데이터 관리)
기준 데이터를 관리하는 메뉴입니다.

**1. 데이터 관리 서브 메뉴**
- **국사 리스트**: 국사 정보(국사명, 코드, 팀명 등)를 추가/수정/삭제합니다.
- **랙 모델**: 랙의 규격(모델ID, 크기, 상징 등)을 관리합니다.
- **대상 설비**: 설비 유형 코드를 관리합니다.

**2. 기타 기능**
- **비밀번호 변경**: 우측 상단 열쇠 아이콘을 눌러 관리자 비밀번호를 변경할 수 있습니다.
- **데이터 내보내기**: 현재 등록된 모든 기준 데이터를 배포용 파일(`initialData.js`)로 추출합니다.

---

## 4. 환경 설정 (프린터)
우측 상단의 **[프린터 설정]** 버튼을 통해 라벨 프린터 연동 설정을 변경할 수 있습니다.
- `Env.Ini` 파일의 내용을 직접 편집합니다.
- 변경 후 **[설정 저장]**을 눌러야 적용됩니다.

---

## 5. 자주 묻는 질문 (FAQ)
**Q. 출력이 되지 않습니다.**
A. 라벨 프린터 드라이버가 정상적으로 설치되었는지 확인하고, [프린터 설정]에서 설정값이 올바른지 확인하세요.

**Q. 랙 모델이 '일치하는 모델 없음'으로 나옵니다.**
A. 관리자 모드의 [랙 모델] 탭에 해당 규격이 등록되어 있는지 확인해 주세요.
